# Market-Research-Customer-Segmentation-Excel
Excel-based Market Research &amp; Customer Segmentation Tool to analyze customer data and identify target segments using dashboards.
📌 Project Overview

This project is an advanced Excel-based Market Research & Customer Segmentation Tool designed to analyze customer data and identify different customer groups.

It helps businesses understand customer behavior and make data-driven decisions.
